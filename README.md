## ✨ Features

### 🔒 Instalock  
Instant champion selection with up to **3 fallback champions**.

### 🔒 Auto Ban  
Automatic champion banning with **3 backup options**.

### 🔄 Auto Accept  
Automatically accepts queue pop-ups in milliseconds.

### 💬 Chat Toggle  
Enable or disable the in-game chat at any time.

### 🔍 Lobby Reveal  
Instantly opens Porofessor analysis upon entering champion select.

### 🎨 Theme Selector  
Manage, import, and switch between custom themes.

## 🛠️ Installation

1. Download the latest release `.zip` file  
2. Extract all files to a folder of your choice  
3. (Optional) Create a desktop shortcut for quick access  
4. Open League of Legends  
5. With the League client running, launch the application  
6. Configure your preferred features

---

## 🧩 Requirements

- Windows 10 or Windows 11  
- League of Legends (any region)  
- .NET 8.0 Runtime or newer
